## 建立的自动化系统

### 1. 资源收集系统
- **技能**: `techart-resource-collector`
- **触发**: 每日 14:00 cron (`daily-resources-14`)
- **流程**: 关键词搜索 → 网站验证 → 内容验证 → 摘要生成 → Git 推送
- **验证机制**: HTTP 状态码、内容长度、技术关键词匹配、代码语法检查
- **输出**: `Toufuart/YYYY-MM-DD/` 目录，带质量报告

### 2. 鬼点子生成系统
- **技能**: `idea-generator`
- **触发**: 每日 10:00 cron (`daily-ideas-10`)
- **流程**: 基于用户背景生成 10 个创意点子
- **分类**: 工具开发、视觉效果、学习项目、跨界融合
- **输出**: `idea-factory/` 仓库，每个点子详细模板

### 3. 问答索引系统
- **技能**: `sleep`
- **触发**: 用户说 "sleep" 或 "执行 sleep 任务"
- **流程**: 上下文审查 → 问答生成 → 资源引用 → 索引更新
- **输出**: 
  - `Toufuart/qa/YYYY-MM-DD.md`（每日问答）
  - `Toufuart/qa/index.md`（总索引）
  - 资源文件末尾的问答引用
