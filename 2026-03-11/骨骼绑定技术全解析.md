### 摘要
骨骼绑定是角色动画的基础，涉及将3D模型的顶点与骨骼系统关联，实现角色的自然运动。本文系统介绍了骨骼绑定的核心概念、工作流程和高级技术，包括权重绘制、IK系统、约束和变形器的应用。

### 技术细节
- **使用的工具/语言**: Maya, Blender, 3ds Max, Python脚本, C++插件
- **涉及概念**: 
  1. **骨骼层级结构**: 父子关系、局部空间与全局空间变换
  2. **蒙皮权重**: 顶点与骨骼的关联强度，平滑过渡区域
  3. **反向运动学(IK)**: 基于目标位置计算关节旋转
  4. **正向运动学(FK)**: 基于关节旋转计算末端位置
  5. **约束系统**: 位置、方向、父子等约束
  6. **变形器**: 晶格、包裹、混合形状等辅助变形

# Maya Python 自动权重工具示例
import maya.cmds as cmds
import maya.mel as mel

def auto_weight_selection(geometry, joints):
    """为选定的几何体和关节自动分配权重"""
    
    # 选择几何体和所有关节
    cmds.select(geometry)
    for joint in joints:
        cmds.select(joint, add=True)
    
    # 执行蒙皮绑定
    skin_cluster = cmds.skinCluster(
        geometry, 
        joints, 
        tsb=True,  # 使用双四元数蒙皮
        mi=3,      # 最大影响数
        omi=True,  # 保持最大影响
        dr=4.0     # 衰减半径
    )
    
    # 平滑权重
    mel.eval('artSmoothSkinWeights;')
    
    return skin_cluster

# 使用示例
character_mesh = 'pCharacter_Geo'
skeleton_joints = ['root_jnt', 'spine_jnt', 'l_shoulder_jnt', 'r_shoulder_jnt']
skin_cluster = auto_weight_selection(character_mesh, skeleton_joints)
