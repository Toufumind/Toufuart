# Docker容器化学习项目

## 项目概述
本项目包含Docker容器化技术的深度学习和实践指南，涵盖从基础概念到生产部署的完整知识体系。

## 内容结构

### 文档资料
1. **`docs/docker-core-concepts.md`** - Docker核心概念深度分析
   - Docker架构组件详解
   - 镜像系统和存储机制
   - 容器运行时和网络系统
   - 安全机制和最佳实践

2. **`docs/docker-installation-guide.md`** - Docker安装配置指南
   - Ubuntu 24.04 LTS安装步骤
   - WSL2特定配置
   - 故障排除和安全建议

3. **`docs/docker-practical-guide.md`** - Docker实践指南
   - 从安装到生产部署
   - 常见问题解决方案
   - 安全最佳实践和监控

### 工具脚本
- **`install-docker.sh`** - Docker自动化安装脚本
  - 支持重试机制
  - 多源下载支持
  - 错误处理和验证

## 学习目标
1. 深入理解Docker架构和工作原理
2. 掌握Docker安装、配置和优化
3. 学习容器安全加固和性能监控
4. 实践项目容器化和生产部署

## 适用人群
- 容器化技术初学者
- 需要部署容器化应用的开发者
- 系统管理员和运维工程师
- 对云原生技术感兴趣的学习者

## 使用说明
1. 阅读核心概念文档建立理论基础
2. 使用安装脚本配置Docker环境
3. 按照实践指南进行实际操作
4. 参考最佳实践优化容器配置
